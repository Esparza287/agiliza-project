Introducción

Los establecimientos del país se encuentran en un proceso de digitalización, el ministerio de educación comenzará a exigir los datos computarizados. Es aquí donde aparece una oportunidad para desarrollar un sistema orientado a liceos y colegios.

Problemática

El Liceo Industrial posee un sistema web para la gestión de sus registros, pero este apenas cubre con las necesidades básicas, es poco intuitivo y no permite el acceso libre a la información que este almacena.

Solución

Se desarrolló un sistema multiplataforma, que satisface en primera instancia los requerimientos del Liceo Industrial, está construido bajo tecnologías web, tales como HTML5, JavaScript & CSS. Adicionalmente incluye una aplicación móvil orientada a dispositivos Android.

Objetivos

Agiliza es una plataforma web y móvil, que se proyecta como un sistema que gestione toda la información académica y otros procesos de las instituciones. Se espera pueda ser implementado en cualquier establecimiento.

Alcances & Limites

Especificación de las funciones que realiza el sistema.

Gestión de Usuarios

La aplicación ofrece una completa gestión para administrar diferentes tipos de usuarios, permitir su acceso mediante credenciales, y controlar el contenido que estos visualizan.

Registros de Alumnos

Agiliza gestiona diferentes datos, tales como, niveles, cursos, horarios, bloques, asignaturas, etc. Estos se enlazan y permite almacenar información de los usuarios, tales como notas, asistencia, entre otros registros.

Módulos Extras

Adicionalmente a la planificación, se desarrollan módulos que cubren las necesidades de otros tipos de procesos de la institución, entre ellos se destacan la gestión de los pedidos de taller, y la biblioteca.

Certificación

Este sistema posee la limitante de no tener acceso a la información que provee el Ministerio de Educación, ya que, para obtener este beneficio, se debe realizar un proceso de certificación.

Tecnologías

Mención de los diferentes lenguajes y herramientas utilizadas en el desarrollo del proyecto.

Aplicación WEB

Agiliza está construida completamente orientada a la WEB, sobre lenguajes como HTML5, JavaScript y CSS, potenciada con frameworks como AngularJS y JQuery en el front-end. El back-end está desarrollado en PHP, diseñado como una API Rest, y esta complementado con Medoo y Slim, para agilizar las consultas a la base de datos MySQL.

Aplicación Móvil

La aplicación para smartphone está desarrollada bajo la plataforma Android, para las versiones 5.0 Lollipop y superiores, ya que posee las últimas tecnologías sugeridas por Google y su API de nivel 21. Su diseño sigue las nuevas directrices de Google, Material Design. Posee un servicio de notificaciones push potenciado por Firebase.

Más utilidades para grandes características.

Desglose de las principales herramientas que potencian nuestro sistema.

AngularJS

Framework MVW basado en JavaScript, creado por Google, que nos permite implementar un sistema "One Page Aplication", gestionar de manera veloz la información, obteniéndola de manera asíncrona, pre-cargar los datos en segundo plano y mucho más, gracias a sus directivas.

Firebase

Una plataforma WEB y Móvil mantenida por Google, diseñada para ayudar a los desarrolladores a construir aplicaciones de alta calidad, y que nos provee de una base de datos en tiempo real, y de notificaciones push.

Jquery Plugins

El framework más utilizado a la fecha, basado en JavaScript, nos facilita utilidades en forma de plugins, que nos ayudan en la implementación de algunos procesos como la gestión de horarios, impresión de informes, animaciones, etc.

Slim & Medoo

Slim y Medoo son micro frameworks que nos ayudan en la creación de una API Rest. Slim mejora la sintaxtis de esta, y Medoo gestiona las consultas que se realizan a la base de datos MySQL. Con esta API Rest logramos una rápida y clara comunicación entre diferentes dispositivos.

Conclusión

Cualquier organización necesita digitalizar sus datos, esto incluye a los establecimientos educacionales, más aún, siempre irán surgiendo nuevos requerimientos de desarrollo en los sistemas, es por ello que consideramos que la principal característica de una aplicación, es su escalabilidad y su capacidad de adaptación a los cambios, Agiliza está desarrollado bajo este paradigma, buscando siempre innovar en sus procesos y cubrir las nuevas necesidades que vayan apareciendo.
